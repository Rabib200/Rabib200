Designed full system architecture. Architected for 8,000-10,000 users with message queues for high-throughput processing. Implemented DevOps with Portainer, EC2, ECR, multi-stage Docker builds (~30% smaller images), and GitHub Actions CI/CD (12 min → 5-6 min).

**Voice2Policy** — AI Civic Engagement Platform
`NestJS` `Next.js` `Docker` `AWS SageMaker` `deBERTa-v2`
Fine-tuned deBERTa-v2 via SageMaker for factual extraction and GPT-based policy analysis. Improved policy review speed by 3x. Supported 1,000 concurrent users.

`React.js` `Next.js` `Node.js` `OpenAI API` `PostgreSQL`
AI-powered calendar that automates booking flows. Reduced booking time by 50%. Scaled to 5,000+ users in the first quarter.

**Dive HR** — Global HR Management
`React.js` `Next.js` `Node.js` `PostgreSQL`
Multi-region session listings, RBAC, smart content filtering. 1,000+ sessions across 15+ countries within 2 months.

**Munchies Platform** — Restaurant Operations at Scale
`NestJS` `TypeScript` `PostgreSQL` `Redis`
